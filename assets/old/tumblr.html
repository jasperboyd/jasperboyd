The Weather Sound

The major goal behind this project was finding a new way to distribute commonly consumed data. So many people everyday log onto sites like Wunderground and look at the strings that will explain to them what it's like outside weather-wise. This certainly effectively communicates the data, but what if the process was simpler? What if that data could be communicated by the speaker built into an alarm clock?

Max/MSP/Jitter allows for easy manipulation of digital audio signals. For this project the challenge was getting the weather data into Max. To solve this problem I developed a java external for Max which uses JSON to get the data from Wunderground.

This project is currently under development but should be complete by April of 2013. If you want to find the most up to date copy of the patch you find it here.

Although this project started as a Max patch, I hope to one day port it to some kind of web application. If you know of a javascript library that allows for audio processing, or alternatively someway to run Max constantly on a server email me.
